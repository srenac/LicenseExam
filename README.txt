License Exam Program

Note: Written using Net Beans IDE in Java

Description: License exam program stores a set of correct answers into an array and a separate array that holds the user's answer input. Program test to see if test taker inputs the correct answers. Console output prompts user to answer each of the 20 questions one by one. The Missed() function returns an array that contains the incorrect questions that were answered. Program keeps track of the correct and incorrect questions and checks to see if the exam score is a 'Pass' or 'Fail' grade. If score is less than or equal to 15 then the user 'Passes' the exam.
